# Regalo Personalizado para Ale 💜

Una página web creada con amor por Rubén para su novio Ale, basada en el análisis detallado de su conversación de WhatsApp.

## Características

- ✨ Diseño moderno y responsive
- 🌓 Tema oscuro/claro con persistencia
- 💫 Animaciones suaves y efectos visuales
- 🎵 Reproductor de música con las bandas favoritas de Ale
- 📖 Carta escrita en el estilo de Rubén
- 📊 Línea temporal de la relación
- 🖼️ Galería de recuerdos (con placeholders)
- 😂 Sección de chistes internos
- ⏱️ Contador de tiempo juntos
- 💌 Mensajes destacados de la conversación
- � Nueva sección de curiosidades sobre la relación
- 💭 Sección de sueños futuros
- 🎮 Referencias a Valorant y gustos musicales
- 💜 Temática morada basada en el emoji favorito de la pareja

## Tecnologías utilizadas

- HTML5 semántico
- CSS3 avanzado (variables, flexbox, grid, animaciones)
- JavaScript vanilla (sin frameworks)
- Diseño responsive
- Animaciones CSS y JS
- localStorage para persistencia de tema

## Cómo ejecutar localmente

1. Abrir el archivo `index.html` en cualquier navegador moderno
2. No se requiere servidor local ni dependencias externas
3. La navegación funciona mediante anclajes en la misma página

## Despliegue en GitHub Pages

1. Subir este repositorio a GitHub
2. Ir a Settings > Pages
3. Seleccionar la rama `main` y la carpeta `/ (root)`
4. GitHub proporcionará una URL donde estará disponible el sitio

## Personalización

Para personalizar aún más el regalo:

1. **Reemplazar placeholders de imagen**: En `script.js`, buscar las URLs de `https://via.placeholder.com/` y reemplazarlas con imágenes reales de la pareja
2. **Agregar audio real**: En `initAudioPlayer()`, reemplazar las URLs de placeholder con enlaces reales a canciones de las bandas mencionadas
3. **Ajustar fechas**: Modificar la fecha de inicio en `initCounter()` si es necesario
4. **Añadir más contenido**: Ampliando las secciones en las funciones de inicialización en `script.js`
